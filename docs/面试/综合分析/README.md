# 综合分析文档索引

> 更新时间：2026-04-19  
> 本目录用于承接项目展示稿、参考面经与项目定制面试问答。当前推荐入口已经切换到 200 题版本。

## 推荐阅读顺序

1. 先读 `../tech_part/README.md`，建立项目技术主线。
2. 再读 `project_interview_questions_200_20260419.md`，按模块进行面试训练。
3. 如果需要补充互联网面经素材，再读 `ai_agent_interview_reference_roundup_20260418.md`。
4. 如果需要项目宣传图谱思路，再读 `agent展示与宣传图谱设计建议_20260417.md`。

## 使用建议

- 如果目标是准备本项目面试，优先把 `tech_part` 和 `200 问` 结合起来讲。
- 回答时不要只讲 why，要同步讲 use what、how to do、代码位置和实际效果。
- 如果面试官追问 RAG、Supervisor、Grounding、Workspace、SSE，就回到 `tech_part` 的对应子文档展开。
